Mass-casualty triage exercise using the START protocol (Simple Triage And Rapid Treatment).

START decision rules (in order):
1. **Can the patient walk?** → GREEN (minor)
2. **Is the patient breathing?** If no, attempt to reposition airway. If still not breathing → BLACK (deceased/expectant). If breathing after repositioning → RED (immediate).
3. **Respiratory rate** > 30/min → RED (immediate)
4. **Capillary refill** > 2 seconds OR no radial pulse → RED (immediate)
5. **Cannot follow simple commands** (mental status) → RED (immediate)
6. **Otherwise** → YELLOW (delayed)

You are at an incident with five casualties. Apply START to each and produce a tag color for every patient.

**Patient 1**: 8-year-old, walking and crying, scrape on knee, asking for parents.

**Patient 2**: Adult male, not walking. After repositioning airway: not breathing. No pulse.

**Patient 3**: Adult female, supine, breathing at 36/min, radial pulse weak but present, follows simple commands.

**Patient 4**: Adult male, walking, holding bleeding forearm, talking to bystanders, no visible major bleeding controlled with direct pressure.

**Patient 5**: Adult female, supine, breathing at 14/min, capillary refill <2 sec, does not respond to "squeeze my hand."

Produce a numbered list with EXACTLY one of `GREEN`, `YELLOW`, `RED`, `BLACK` for each patient. Then briefly justify each in one sentence citing which START rule triggered the color.

Format:
```
Patient 1: GREEN — reason
Patient 2: BLACK — reason
...
```
