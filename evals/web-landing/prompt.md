Build a polished product landing page as a SINGLE self-contained `index.html` file.

Requirements:

- **Sticky navigation bar** at the top (stays visible on scroll) with the product name and a few nav links.
- **Hero section** with a large headline, a short supporting subheadline, and a prominent call-to-action (CTA) button.
- **Features section** with exactly three feature cards (icon/title/description each), laid out in a row on desktop and stacked on mobile.
- **Footer** with copyright and a couple of links.

Constraints:

- Everything in ONE file: `index.html` in the workspace root.
- All CSS goes inline in a `<style>` block in the `<head>`. No external stylesheets, no CSS/JS frameworks (no Bootstrap/Tailwind/etc.), no build step, no CDN.
- **Responsive**: must look good on both mobile and desktop. Use a `@media` query and a modern layout (CSS `flex` or `grid`).
- **Tasteful, modern CSS**: use gradients, generous spacing, rounded corners, and hover states (e.g. on the CTA button and the feature cards).
- **Semantic HTML5**: use `<nav>`, `<header>`/hero, `<section>`, and `<footer>` elements with real headings.

Write the file as `index.html` in the workspace root. Do not create any other files. Do not run it.
