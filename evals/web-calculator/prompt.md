Build a working calculator web app in ONE file named `index.html`, written to the workspace root.

Requirements:
- A button grid plus a display area.
- The four arithmetic operators: `+`, `-`, `*`, `/`.
- Decimal point support.
- A clear button.
- Keyboard input (digits, operators, Enter/`=`, Escape/clear, Backspace).
- CORRECT arithmetic and operator precedence for chained input (e.g. `2 + 3 * 4` must evaluate to `14`, not `20`).
- Styled and visually presentable.
- Responsive to both mouse clicks and key presses.

Everything must be inline in the single `index.html`: HTML, CSS in a `<style>` block, and JavaScript in a `<script>` block. No external dependencies, no CDN links, no build step. The file must run by simply opening it in a browser.

Write the file as `index.html` in the workspace root.
