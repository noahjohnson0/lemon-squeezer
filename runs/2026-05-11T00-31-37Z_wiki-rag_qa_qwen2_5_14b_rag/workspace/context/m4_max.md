# Apple M4 Max

The Apple M4 Max is a system on a chip designed by Apple Inc., announced on October 30, 2024. It is the highest-end variant of the M4 family, alongside the standard M4, M4 Pro, and M4 Ultra. The chip is fabricated on TSMC's second-generation 3 nm process (N3E).

The M4 Max contains either 14 CPU cores (10 performance + 4 efficiency) or 16 CPU cores (12 performance + 4 efficiency) depending on the configuration, and either 32 or 40 GPU cores. It has a 16-core Neural Engine and a 546 GB/s unified memory bandwidth. The chip supports up to 128 GB of unified memory.

The M4 Max debuted in the 14-inch and 16-inch MacBook Pro models. Apple advertises the chip as suitable for professional workflows including video editing, 3D rendering, and large language model inference, with the unified memory architecture allowing larger models to fit entirely on-chip without the need to swap to disk.

For local LLM inference on Apple Silicon, the unified memory architecture means the GPU can access the full system RAM directly. A 48 GB M4 Max can comfortably run 70B parameter models at Q4 quantization (~40 GB on disk), enabling near-frontier inference on a laptop without external GPU.
