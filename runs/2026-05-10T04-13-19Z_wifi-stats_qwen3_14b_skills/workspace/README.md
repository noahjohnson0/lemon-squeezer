# Wi-Fi Stats App

This app displays Wi-Fi network statistics on macOS using a FastAPI backend and Next.js frontend.

## Getting Started

### Backend (Python)
1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Run the FastAPI server:
   ```bash
   uvicorn main:app --reload
   ```

### Frontend (Next.js)
1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

Visit http://localhost:3000 to view the Wi-Fi stats dashboard.
